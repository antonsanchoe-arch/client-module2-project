## User Stories

**NOTE -**   all the actions a user can do in the app.

-homepage – As a user, I want to access the homepage to see what the app is about.
-characters list – As a user, I want to see all existing characters displayed on cards.
-characters create – As a user, I want to create a new character through a form.
-characters edit – As a user, I want to edit an existing character and keep the previous values.
-characters delete – As a user, I want to delete any character easily from the list.
-characters search – As a user, I want to filter and search characters by name, universe, species, or alignment.
-404 – As a user, I want a friendly error page when I reach a non-existing route.

## React Router Routes (React App)
------------------------------------------------------------
 PATH               | PAGE / COMPONENT        | DESCRIPTION
------------------------------------------------------------

 /                  | HomePage                | Home screen of the app
------------------------------------------------------------
 /characters        | CharactersPage          | Shows all characters
------------------------------------------------------------
 /create            | CreateCharacterPage     | Form to create a new character
------------------------------------------------------------
 /edit/:id          | EditCharacterPage       | Edit an existing character
------------------------------------------------------------
 /search            | SearchPage              | Search + filters for characters
------------------------------------------------------------
 *                  | NotFoundPage            | 404 page for invalid routes
------------------------------------------------------------
